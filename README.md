# 简介

这是存我的各种配置文件，设置等，用来备份，便于快速搭建和回复开发环境。

开发基本都在wsl里开发了，win不安很多环境了。win主要安个scoop当包管理器了。

默认发行版是ubuntu22.04，可能会使用apt，基本问题不大，会使用zsh，因此建议先安装zsh。

# 使用方法已经介绍

## linux
1. 直接使用bash执行,脚本内部有sudo了，有些命令需要读取账号文件夹，和账号有关，避免使用sudo直接执行。

比如`sudo go install`也是错误用法。

2. 使用软链接,迁移已有配置方式:
  - 在仓库新建目录
  - 使用`mv`迁移已有配置文件，可选`cp`进行备份
  - 使用`ln -s`建立软链接

```
bash ***.sh
```
